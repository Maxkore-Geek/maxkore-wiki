## 🚀 快速开始

### 本地预览

```bash
# 克隆仓库
git clone https://github.com/Maxkore-Geek/maxkore-wiki.git
cd maxkore-wiki

# 安装 docsify
npm i -g docsify-cli

# 启动预览
docsify serve docs
```

访问 http://localhost:3000

### 贡献内容

```bash
# 1. Fork 仓库
# 2. 创建新分支
git checkout -b new-content

# 3. 添加或修改文档
# 4. 提交更改
git add .
git commit -m "添加新内容"
git push origin new-content

# 5. 创建 Pull Request
```

## 📝 使用技巧

### Typora 快捷键

| 快捷键             | 功能     |
| ------------------ | -------- |
| `Ctrl + 1~6`       | 标题级别 |
| `Ctrl + B`         | 粗体     |
| `Ctrl + I`         | 斜体     |
| `Ctrl + K`         | 插入链接 |
| `Ctrl + Shift + I` | 插入图片 |
| `Ctrl + Shift + C` | 代码块   |

### Docsify 快捷键

| 快捷键 | 功能     |
| ------ | -------- |
| `s`    | 搜索     |
| `→`    | 下一页   |
| `←`    | 上一页   |
| `t`    | 返回顶部 |
